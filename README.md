# 📄 Resume Screening AI System

## 📌 Overview
This project is an AI-based Resume Screening System that compares a resume with a job description and calculates a match percentage using Natural Language Processing (NLP) techniques.

---

## 🚀 Features
- Text preprocessing
- Stopword removal
- TF-IDF Vectorization
- Cosine Similarity calculation
- Match percentage output

---

## 🧠 Tech Stack
- Python
- Scikit-learn
- NLP
- TF-IDF
- Cosine Similarity

---

## ▶️ How to Run

1. Clone the repository:
   git clone <your-repo-link>

2. Install dependencies:
   pip install -r requirements.txt

3. Run the program:
   python main.py

---

## 📊 Output
The system calculates how much a resume matches a job description in percentage.
